## RuneScape Terminology

| Term | Description |
|------|-------------|
| 2h | Two-handed Weapon |
| Abil | Ability |
| Aggro | Aggression / NPC Aggression |
| Alt1 | A pixel-based screen reader that provides several APIs to help create plugins |
| AoE | Area of Effect |
| Apot | Adrenaline potion |
| Auto | Autoattack |
| Base | Base tank / Main tank |
| Basic | Basic ability |
| BoB | Beast of Burden |
| BiS | Best-in-slot - The best equipment in-game |
| Build | Gain adrenaline |
| Cade | Barricade - defensive ability that reduces most damage taken to 0 |
| Cept | Intercept - Spell that allows you to redirect incoming damage from another player |
| CD | Cooldown |
| Crit | Critical Strike |
| DD | Death dot (stand on the same tile) |
| Devo | Devotion - defensive ability that makes protection prayers 100% effective |
| DoT | Damage over time ability |
| DPM | Damage per minute |
| DPS | Damage per second (literal), attack as hard as possible (command) |
| DW | Dual wield weapons |
| EGWD | Elder God Wars Dungeon (also referred to as GWD3) |
| Flick | Prayer flicking (swapping between overhead prayers) |
| GWD | God Wars Dungeon |
| GWD2 | The Heart of Gielinor (also referred to as God Wars Dungeon 2) |
| GWD3 | Elder God Wars Dungeon |
| HP | Hitpoints / Lifepoints |
| Immort | Immortality - defensive ability that revives you if you die |
| NPC | Non-playable character |
| MD | Melee distance - standing within melee range of your target |
| MH | Mainhand (equipment slot) |
| Ovl | Overload - stat boosting potion |
| Proc | Random effect activation |
| Res/Reso | Resonance - defence ability that heals you for the lifepoints you would have taken from a damaging attack |
| Revo | Revolution - system that automatically triggers combat abilities |
| Spec | Special attack |
| SS | Soul Split - high level overhead prayer that heals you for a percentage of your damage dealt |
| Sticks | Incense sticks |
| TC | Target cycle |
| TP | Teleport |
| Thresh | Threshold ability |
| Tick | Game tick or cycle - period of 0.6 seconds that RuneScape runs on |
| Vuln | Vulnerability - Debuff that increases damage dealt by 10% for 60 seconds |
| Ult | Ultimate ability |
| Voke | Provoke - Defence ability that draws aggression of your target from a teammate |
| Xlog | Forcibly closing client |
